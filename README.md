# The Go Programming Language

## Descrição
Um projeto desenvolvido em Go.

## Instalação
Para instalar o projeto, siga os passos abaixo:

```sh
git clone https://github.com/danielbaldoino/go-dev.git
cd go-dev
go mod tidy
```

## Uso
Para executar o projeto, utilize o comando:

```sh
go run main.go
```
